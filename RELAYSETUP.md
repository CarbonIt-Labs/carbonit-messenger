# Tutorial on Relay setup for anyone New.

## Follow the steps, ignore the dependencies for now.
Dependencies: 

1. Python 3.13+
   ```
   sudo apt install python<version>
   ```
   example: sudo apt install python3.13 (No spaces.)

2. aiohttp-socks (SOCKS connector for asyncio)
   ```
   pip install aiohttp-socks
   ```
2. tor


# Steps to set-up a Carbon relay

1. Setup a device with Ubuntu server LTS on it.
2. WHile setup, You could name the Relay as 'carbonit_relay' or 'carbonitrelay'.
3. After Setup, use sudo apt update and run sudo apt upgrade if any upgradables are found.
